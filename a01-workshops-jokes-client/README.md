## Jokes client

Write app which get multiple jokes changed every prediefined period of time

You can use API for jokes:

http://api.icndb.com/jokes/random


Example: `./joker -n 3 -i 20s`

```
+------------------------------------------------+
|                                                |
| Joke 1                                         |
| Joke 2                                         |
| Joke 3                                         |
|                                                |
|                                                |
|                                                |
+------------------------------------------------+

after 20s

+------------------------------------------------+
|                                                |
| Joke 4                                         |
| Joke 5                                         |
| Joke 6                                         |
|                                                |
|                                                |
|                                                |
+------------------------------------------------+

etc

```
